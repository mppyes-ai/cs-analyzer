### 5. 问题汇总

1. **连接池警告**：Connection pool is full, discarding connection: localhost
   - 影响：可能导致部分请求延迟
   - 建议：增加连接池大小或限制并发数

2. **单批次处理时间较长**：300-320秒/批次
   - 影响：大批次处理时间较长
   - 建议：优化批次大小或模型响应速度

### 6. 优化建议

1. **增加连接池大小**：避免连接池满导致的延迟
2. **优化批次大小**：根据模型响应速度调整批次大小
3. **监控缓存命中率**：持续监控并优化缓存策略
4. **限制并发数**：避免过多并发导致系统资源紧张
