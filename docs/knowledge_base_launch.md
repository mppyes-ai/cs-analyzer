## 使用说明

### 访问方式
1. 启动Streamlit: `streamlit run 🏠_主页.py`
2. 侧边栏导航: "知识库首页"
3. 快捷入口: 规则审核、规则录入、知识检索、效果分析

### 操作流程
1. **录入规则**: 知识库首页 → 规则录入 → 填写表单 → 保存
2. **审核规则**: 知识库首页 → 规则审核 → 查看详情 → 批准/拒绝
3. **检索规则**: 知识库首页 → 知识检索 → 输入关键词 → 查看结果
4. **分析效果**: 知识库首页 → 效果分析 → 查看统计图表
