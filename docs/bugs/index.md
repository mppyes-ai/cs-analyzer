## Bug ID命名规则

格式：`[ID]-[日期].md`

- **ID**: `[类型]-[架构层]-[序号]`
  - 类型: C(ritical) / H(igh) / M(edium) / L(ow) / S(ecurity) / P(erformance)
  - 架构层: CORE / DB / API / MSG / CFG / UI
  - 序号: 001-999
- **日期**: `YYYYMMDD` (8位数字，无连字符)

**示例**: `H-UI-011-20260403.md`
